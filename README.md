# matplotcr

A Crystal wrapper for matplotlib.

# Features

* Line plots
* Scatter plots
* Histograms
* Line segments
* Axis (horizontal and vertical) lines
* Font customisation
* Annotations
* Custom size and DPI
* Sub-plots
* Horizontal and vertical limits
